## What You'll Learn

- **Basics** — `init`, `add`, `commit`, `status`, `log`
- **Branching & Merging** — branches, merge, rebase, conflict resolution
- **Collaboration** — remotes, pull requests, forks, code review
- **Advanced** — stash, bisect, reflog, hooks, submodules
- **Automation** — GitHub Actions, CI/CD pipelines, release workflows
