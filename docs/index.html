Learning Roadmap

Follow this structured path from absolute beginner to Git master.

Week 1
Git Basics
Installation, configuration, first commit, staging, viewing history. Understand what version control is and why it matters.
Beginner 7 days
Week 2
Branching & Merging
Create and switch branches, merge changes, resolve conflicts. Master the branch-based workflow that makes Git powerful.
Beginner 7 days
Week 3
Collaboration
Remote repositories, pull requests, fork workflow, code review. Learn how teams collaborate effectively using GitHub.
Intermediate 7 days
Week 4
Advanced Topics
Stashing, rebasing, cherry-pick, tags, GitHub Actions, CI/CD pipelines. Take your skills to production level.
Advanced 7 days
